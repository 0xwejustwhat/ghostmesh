# Code Of Conduct

Ghost Mesh contributors are expected to work with care, curiosity, and respect.

## Expected Behavior

- Be constructive and specific.
- Assume good intent while addressing impact.
- Prefer clear technical arguments over status or authority.
- Make space for new contributors.

## Unacceptable Behavior

- Harassment, threats, or discriminatory language.
- Personal attacks.
- Publishing private information without consent.
- Repeated disruption of project work.

Project maintainers may remove comments, issues, pull requests, or contributors that violate these expectations.
