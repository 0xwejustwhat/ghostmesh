# Roadmap

## MVP Core

- Phase 0-9 open-source runtime foundation.
- Durable Card runtime with Postgres.
- Worker SDK and human validation surface.
- Shadow lanes, mutation gates, and promotion.
- Boundary adapters and operator read models.
- Public-ready documentation and agent-facing skill templates.

## Near-Term

- Richer acceptance contract rules.
- More boundary adapter examples.
- More complete dashboard UI.
- OpenTelemetry traces and metrics export.
- Hardened Helm chart and production deployment guidance.

## Later

- Hosted enterprise control plane.
- Advanced policy and audit features.
- Marketplace-style workflow and worker distribution.
- Deeper MCP edge adapter library.
